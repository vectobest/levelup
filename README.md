# 🎮 LevelUp - Gamified Habit Tracker

LevelUp is a full-stack MERN application that helps users build habits using a gamified system with XP, streaks, and levels.

# LevelUp Habit Tracker 🎮

Live Website:  
https://levelup-nu-ten.vercel.app

Backend API:  
https://levelup-backend-a659.onrender.com

## 🛠 Tech Stack

Frontend
- React (Vite)
- TailwindCSS
- Axios

Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

Deployment
- Vercel
- Render
- MongoDB Atlas

## ✨ Features

- User Authentication
- Habit Tracking
- XP Level System
- Streak Tracking
- Avatar Evolution
- Delete Habit
- Gamified Dashboard
